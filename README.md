### 前端 : HTML + CSS + Javascript
### 後端 : Node.js + Express.js + MySQL

[Node.js 下載網站連結](https://nodejs.org/en/?source=post_page-----317beefdf182--------------------------------)

### 安裝套件
```
// 在終端機輸入
npm init
npm install express
npm install body-parser
npm install multer
npm install mysql
npm install nodemon
```

### 啟動專案
```
nodemon server.js
```
